# Music Mentor

Music Mentor is a modern React + Vite student project for learning musical instruments.

## Features
- Home page with hero section, explanation, featured instruments, benefits, and testimonials
- Instrument pages for Guitar, Piano, Violin, Drums, and Ukulele
- Lessons with search and filters
- Working lesson modal with "Start Lesson" and "Mark as Completed"
- Practice tracker with total minutes, completed lessons, streak, and session history
- AI-style simulated features:
  - practice tips
  - lesson simplifier
  - motivation box
- Responsive UI with a dark blue / purple visual style

## Run locally
```bash
npm install
npm run dev
```

## Tech
- React
- Vite
- React Router
- Lucide React
- Framer Motion (installed if you want to extend animations later)
